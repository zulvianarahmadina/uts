MATERI

Praktikum sistem Informasi Geografis berbasis WEB merupakan pembelajaran mengenai tentang bagaimana pengolahan dan pengaplikasian sistem informasi geografis diolah dalam bentuk WEB. Mata kuliah Praktikum SIG Berbasis Web membahas mengenai bagaimana data dan informasi geospasial ditampilkan melalui internet, baik pada platform berupa desktop, mobile maupun media lainnya. Mata kuliah ini dimulai dari pembahasan mengenai cara kerja internet dan halaman web, konsep dan cara kerja client dan server pada WebGIS, arsitektur WebGIS serta desain dan evaluasi WebGIS. Mahasiswa akan dikenalkan berbagai bahasa pembentuk webgis, diantaranya yaitu HTML, CSS, Java Script, Bootstrap, Leaflet JS dan lain sebagainya
